# Saúde Mental

## Descrição
Este é um projeto simples de uma página web feita em HTML e CSS sobre saúde mental.

A página mostra uma mensagem para jovens que estão sem motivação ou com muitos pensamentos, incentivando a organizar a mente e melhorar a vida.

## ODS da ONU relacionada
ODS 3 – Saúde e Bem-Estar

## Tecnologias usadas
- HTML
- CSS
